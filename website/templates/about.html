The Goal was to develope coding skills using a practial project.
Choosing a project was an interesting process, however after considering a few options I settled on a quiz type site which can be adapted in the future.

The whole process is a learning curve and I'm pretty sure that I am very close to the bottom of the curve at the moment.
I have encountered many issues in my coding journey so far and somehow found solutions for most.
There are a few bugs which will need to be addressed in the furture however at this point I'm happy enough to release this project into the wild and see how it works.

If you would like to leave some feedback you can contact me by e-mail at alan855347ryan@gmail.com
